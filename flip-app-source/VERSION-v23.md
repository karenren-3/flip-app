**发布日期:** 2026-03-09  
**版本代号:** Ashes & Embers

## 🎯 核心特性

### 1. 极高景深卷边
- `perspective: 2000px` - 近大远小透视感极强
- `rotateY(60°→110°)` - 大幅旋转弧度
- `rotateZ(0°→20°)` - 纵向扭曲增强立体感

### 2. 灰飞烟灭动画
```javascript
exit={{
  x: x.get() > 0 ? 400 : -400,
  y: -100,              // 向上飘散
  rotateY: x.get() > 0 ? -110 : 110,
  rotateZ: x.get() > 0 ? 20 : -20,
  scale: 0.2,           // 迅速缩小至 20%
  opacity: 0,
  transition: { duration: 0.7, ease: "circOut" }
}}
```

### 3. 31 天治愈金句库
每日自动轮换，全月不重样。

### 4. 三向手势交互
| 方向 | 效果 | 反馈词 |
|------|------|--------|
| 右滑 | 圆满 | 释怀/放下/新生/圆满/过关/翻篇 |
| 左滑 | 暂存 | 随机反馈词 |
| 上滑 | 释怀 | 进入呼吸仪式 |

### 5. 呼吸仪式流程
```
滑空所有卡片
    ↓ (600ms)
暖心告别语 (3.5 秒)
    ↓
3-2-1 呼吸倒计时 (2 秒/次)
    ↓
晚安定格
```
